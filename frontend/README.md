## Installation

```bash
npm install
```

- Create .env file

```bash
cp .env.example .env
```

> **Note**
> You can use already deployed backend part, just set: REACT_APP_BACKEND_URL=https://back.blockaids.online

- Running the app

```bash
npm start
```

- Running tests in the interactive watch mode

```bash
npm test
```

- Building the app for production

```bash
npm run build
```
